Flips the bit at the index bit_idx.

If the bit is set, it's unset. If the bit is unset, it's set. Does not check whether bit_idx is a valid index in the bitarray.

Parameters

bit_idxthe index of the bit to be flipped. Must belong in the interval [ 0, bitarray_length(ba) ).
